# Travnook Sales Quick Reference Card

## Conversation Flow
1. **Greet** → Introduce yourself, confirm service needed
2. **Price** → Share price + document list + timeline
3. **Collect** → Gather docs, verify each, follow up on missing
4. **Pay** → Bank transfer or payment link (+25 AED card fee)
5. **Submit** → Confirm submission, provide timeline
6. **Deliver** → Share visa, congratulate, ask for referrals

## Objection Responses
| Objection | Response |
|-----------|----------|
| Too expensive | Highlight value, speed, approval rate |
| Need time | No rush, keep file ready, follow up 24hrs |
| Comparing agencies | Track record since 2015, 2.5M+ customers |
| Missing documents | Guide client on how to get them |

## Follow-up Timing
| After | Wait | Then |
|-------|------|------|
| Pricing sent | 24h | Gentle check-in |
| Partial docs | 24h | Remind specific doc |
| Payment pending | 24-48h | Friendly nudge |
| Submitted | 2-3 days | Status update |
| No response | 48h → 72h | Final reminder → stop |

## Escalate When
- Refund request
- Rejection
- Discount > 10%
- Client threats
- Previous refusal history
